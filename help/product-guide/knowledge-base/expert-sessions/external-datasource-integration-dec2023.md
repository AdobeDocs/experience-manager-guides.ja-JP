# AEM Guidesの外部データソース統合：仕組み？

多くの大規模な顧客は、複数のデータベースにデータを分散していますが、データを処理するためのシングルソースとして、AEM guidesにデータを統合する必要が生じることがよくあります。
Adobe Experience Manager Guidesには、JIRA、PIM、SQL Data Bases、PLMなどの外部データソースにすぐに利用できるコネクタが追加され、真のContent Hubとなりました。


## セッション録画

## ハイライト

- **日付** - 2023年12月14日
- **参加者の数** - 80
- **期間** - 60分
- **Audience** – お客様、パートナー、およびAdobeの従業員

## 重要ポイント

このセッションでは、次の項目について説明しました。
- AEM Guidesでのデータソースとその使用方法の概要
- 様々なデータソースの接続の設定方法
   - データソーステンプレートでのVelocity ツールの使用
   - マッピングテンプレートの操作
- Web エディターを介してコネクタから取得したデータを表示する
- AEMから取得したコンテンツを
   - トピックジェネレーターを使用したトピックの作成
   - データソースからコンテンツスニペットを挿入する
- AEM guides統合に関するエンドツーエンドの実用的なデモ
   - jiraで利用可能な変更点として、web サイトとPDFのリリースノートを更新する
