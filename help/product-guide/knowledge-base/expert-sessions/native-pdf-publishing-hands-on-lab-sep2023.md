# ネイティブ PDF パブリッシング &quot;Hands on lab - BYOS&quot; - インタラクティブ学習

この「ハンズオン ラボ - BYOS」セッションでは、ネイティブ PDF パブリッシングの詳細な探索を提供します。 様々なユースケースを検討し、主な機能を活用し、Adobeのエキスパートと直接エンゲージして、特定の問題提起や要件に対処します。

## セッションの概要

このセッションでは、PDFのネイティブ公開に関する貴重なインサイトを得ることができます。PDF出力のメタデータの使用から高度なトラブルシューティング手法まで、幅広いトピックを取り上げます。 パブリッシング機能を強化するために、AdobeのエキスパートとネイティブPDFの最近の開発を理解し、ユースケースや特定の要件について話し合います。

## 重要ポイント

このセッションでは、ネイティブ PDF パブリッシングに関する次の内容について説明しました。

- 出力でアセットまたはDITA メタデータを使用します。
- ノート、表、画像、脚注エレメントの操作。
- 画像と表に自動番号を付ける。
- 表、図、見出し、段落への相互参照。
- 変数を使用して、ローカライズされた出力用にDITA オブジェクトを設定します。
- ページの書式設定、スタイル設定、条件付き透かしのオプションなど。
- 中級HTMLを使用したトラブルシューティングの方法。
- PDFのネイティブエンジンへの移行。
- カーマニュアルのオプション/機能。
- 言語変数、再設計されたCSS エディターなどの最近の開発。
