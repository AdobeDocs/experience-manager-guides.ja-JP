# &quot;Delta&quot; コンテンツの移行

これは、AEM Guidesのコンテンツの移行に関するシリーズの3回目のセッションです。
このセッションでは、ある環境から他の環境（オンプレミスからオンプレミス、クラウドからクラウド）への「差分」コンテンツ移行の推奨される手順について説明します。
差分の移行は、同じ基本コンテンツを持つ2つの既存の環境間でコンテンツを転送することであり、以前に移行したコンテンツに変更を同期するために移行が必要であることがわかっているので、対応するシナリオがカバーされます。

- 議論されたユースケースに基づいてコンテンツを移行できるツール
