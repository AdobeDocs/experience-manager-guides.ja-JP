# デプロイメントオプションとCloudへの移行

セッションシリーズでは、AEM Guidesで利用可能なデプロイメントオプション（クラウド以外およびクラウド）とアプリケーションバリエーション（UUID以外、UUID）の概要を説明します。
また、UUID以外のコンテンツをUUIDに移行する手順と、AEM Guides オンプレミス/AMS （uuid）からAEMaaCS上のAEM Guidesにコンテンツを移行する手順について説明します。

## 重要ポイント

このセッションでは、次の内容の概要を説明しました。
- AEM Guidesのデプロイメントオプションとバリエーションについて
- クラウド以外からクラウドに移行するための利用可能な移行オプションの概要
ガイド（uuid）をクラウドに移動する手順は次のとおりです。
- 前提条件：
   - クラウドへの移行の準備状況（クラウドに導入された変更点の確認、実装に関するベストプラクティスアナライザー）
   - 技術的適合性
- 標準ツールを使用したコンテンツの移行：コンテンツ転送ツール
- Guides コンテンツを移行する際に知っておくべき主な設定。
- コンテンツの移行中のトラブルシューティング
- 使用可能なドキュメント
