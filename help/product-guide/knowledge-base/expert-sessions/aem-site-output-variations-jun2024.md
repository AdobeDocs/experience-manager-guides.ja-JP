# セッションの概要

web エクスペリエンスのレイアウトが明確に定義されていれば、アプリケーションのブランディングにも一貫性があります
デジタル全体の統一性や一貫性を維持しながら
エクスペリエンス/ページです。
これらはすべて、Adobe experience manager Guidesが提供するAEMサイト出力の様々なバリエーションで実現できます。
このセッションでは、AEM Sites出力のさまざまなバリエーションを生成する方法について説明します。

## セッション録画

>[!VIDEO](https://video.tv.adobe.com/v/3430649/)

## 重要ポイント

このセッションでは、次の項目について説明します。

- AEMのさまざまなテンプレートタイプのサービス
- テンプレートや公開メカニズムにもとづいたバリエーションの作成方法
- 各テンプレート/メカニズムの使い方の違い
- 既存のAdobe Experience Manager Sites テンプレートを使用して、Adobe AEM Sitesの出力を構築
