# 2026 年 2 月リリースの製品トレーニングおよび学習コンテンツ

このリリースノートでは、製品トレーニングおよび学習コンテンツの 2026 年 2 月リリースで修正された新機能の機能強化と問題について説明します。

## 新機能の強化

次の機能は、製品トレーニングおよび学習コンテンツの 2026 年 2 月リリースで導入されました。

- **字幕のサポート**:「ファイルプロパティ **の新しい「** 字幕を追加 **」オプションを使用して、学習コンテンツに字幕を追加できるようになりました**。 これにより、コースコンテンツ全体が明確になり、検索性が向上します。

  詳しくは、[ 学習コンテンツへのタイトルとサブタイトルの追加 ](../learning-content/lc-basic-blocks.md#add-title-and-subtitle-to-learning-content) を参照してください。

  ![](assets/add-subtitles.png)

- **最終的な負のスコアを有効または無効にする**: クイズプロパティの設定時に、「**負の最終的なスコアを許可**」オプションを使用して負のスコアを制御できます。 これを有効にすると、負のマークが適用されている場合でも、学習者は最終的な最小スコアである 0 を受け取ります。 これにより、スコアがゼロを下回ることがなく確実になることで、学習者のモチベーションが維持されます。

  詳しくは、[ クイズプロパティ ](../learning-content/quiz-properties.md) を参照してください。

  ![](assets/negative-scores-lc.png)

- **右クリックでウィジェットを削除**：クイズ質問を削除する以外に、**右クリック/項目を削除** を使用して、アコーディオン、フリップカード、タブなどのウィジェットを削除できるようになりました。 この機能強化により、既存の *質問を削除* 機能がウィジェットに拡張されました。これにより、クリックが少なく、ナビゲーションが最小限でウィジェットを削除できます。

  詳しくは、[ インタラクティブウィジェットの使用 ](../learning-content/lc-widgets.md) を参照してください。

  ![](assets/delete-widget-items.png)
- **回答の選択肢をピン留め**:SCORM 出力生成中に回答がランダム化された場合でも、特定の回答の選択肢をピン留めして、位置が変更されないようにできるようになりました。 これは、*上記すべて* または *上記なし* などのオプションで特に便利です。

  ![](assets/pin-question.png)
- **短い回答タイプ**：短い回答の質問タイプを使用すると、学習者は事前に定義されたオプションを選択する代わりに、簡単な説明的な英数字の回答を使用して応答できます。 この質問タイプは、学習者が自分の言葉で自分の理解を積極的に思い出して明確にすることを促し、学習者の評価をより魅力的にします。

  詳しくは、[ 質問タイプ ](../learning-content/quiz-insert-questions.md#question-types) を参照してください。


  ![](assets/short-answer.png)
- **クイズ質問の連続試行**:SCORM 出力プリセットの「**学習者はすべての質問を続行するように試みる必要があります**」オプションを使用して、SCORM 出力に対して連続したクイズ試行を強制できるようになりました。 これを有効にすると、学習者は次の質問に移動する前に各質問に回答する必要があります。現在の質問が完了するまでナビゲーションが制限されます。 これにより、ガイド付きの段階的な評価フローと一貫した学習体験が保証されます。

  詳しくは、[SCORM 出力プリセットの設定 ](../learning-content/config-scorm-preset.md) を参照してください。

  ![](assets/scorm-general-tab-v3.png)

## 修正された問題

以下の問題は、製品トレーニングおよび学習コンテンツの 2026 年 2 月リリースで修正されました。

- SCORM 出力を公開して ALM にデプロイすると、L2 クイズ・レポートに、複数回の試行とランダム化された質問バンク選択を使用するクイズの合計スコアと最大スコアが間違って表示されます。 （GUIDES-38855）
- クラウドサーバーで任意のコースを生成すると、`coralui3.css` のスタイルシートが原因で著作権のフッターの下に意図しない空白が表示され、レイアウトの不整合が発生します。 （GUIDES-38853）
- キーボードを使用してアコーディオン付きの学習コースに移動すると、「+」記号やタブタイトルがハイライト表示されず、アクティブな要素を視覚的に識別できません。 （GUIDES-38852）
- SCORM カーボン・テンプレートまたはデフォルト・テンプレートを使用して生成されたコースを横長モードでモバイル・デバイスからアクセスすると、目次（コース・メニュー）にモジュール・リンクが表示されず、ナビゲーションを妨げる問題が発生する。 （GUIDES-38851）
- Experience Manager Guidesでコースの階層を複製する場合、オブジェクトレベルの追加はサポートされていないので、学習目標を作成するには、まず学習グループを作成する必要があります。 （GUIDES-38849）
- オプションが Tab キーまたは矢印キーによってナビゲーションが妨げられるので、キーボードを使用して「次の質問タイプに一致」のドロップダウンオプションにアクセスしようとすると、失敗します。 （GUIDES-38985）
- 見出しスタイルプリセットを適用すると、選択したテキストが消えます。これは、フォントの色が白に変わり、テキストが選択できなくなり、表示されないことが原因で起こる可能性があります。 （GUIDES-39981）
- Mozilla Firefox でExperience Manager Guidesを使用する場合、反転カードは、反転後に前面のテキストを裏側に逆に表示します。 （GUIDES-39983）
- コースの左側のパネルで目次（TOC）をクリックすると、クイズが失敗した場合でも、コースには引き続き完了ステータスが表示されます。 （GUIDES-40398）
- ALM でクイズの次の質問タイプに誤って一致させようとすると、選択したオプションがレポートに表示されません。 （GUIDES-38640）
- PDF出力を生成する際、適用されたオーサリングスタイルが保持されず、その結果、デザインに不整合が生じます。（GUIDES-38642）
