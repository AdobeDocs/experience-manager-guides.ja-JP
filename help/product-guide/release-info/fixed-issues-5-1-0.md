# 5.1.0 リリース（2025年9月）で修正された問題

この記事では、Adobe Experience Manager Guidesの5.1.0 リリースの様々な領域で修正されたバグについて説明します。


新機能と機能強化について詳しくは、[&#x200B; 5.1.0リリースの新機能](whats-new-5-1-0.md)を参照してください。

5.1.0 リリース [&#128279;](upgrade-instructions-5-1-0.md)の アップグレード手順について説明します。


## オーサリング

- ネイティブ PDF テンプレートの&#x200B;**CSS**&#x200B;および&#x200B;**ページレイアウト**&#x200B;のファイルでは、ファイルのロック動作に一貫性がなく、ファイルがロックされている場合でも編集が可能です。 （GUIDES-26688）
- 左側のパネルにカスタムボタンを追加した後にページを更新すると、重複したタブが作成されます。 （GUIDES-30899）
- トピックの`code block`要素内に角括弧（&lt;/または/>など）を含むXML コンテンツを追加すると、最終的な出力でコードブロック要素が空白で表示されます。 （GUIDES-31007）
- ファイルをエディターツールバーからチェックアウトしてチェックインすると、グローバル変数`canCheckIn`および`canCheckOut`の値が正しく設定されません（GUIDES-29890）。
- マップビューでDITA マップを選択すると、マップの子ノードが選択されていないため、最初に選択回数が正しく表示されません。 正しい選択数とすべての子ノードの包含は、その後の選択でのみ反映されます。 （GUIDES-25663）
- **DITA トピック** フィルターを使用してリポジトリーパネルで検索しても、マークダウンファイルは取得されません。 また、**検索と置換**&#x200B;は、Markdown ファイルおよび関連コンテンツでは機能しません。 （GUIDES-27133）
- 拡張機能フレームワークを使用して、エディターツールバーの&#x200B;**メニュードロップダウン**&#x200B;をカスタマイズできません。 その結果、既存のボタンを非表示または表示したり、メニュードロップダウンに新しいボタンを追加したりすることはできません。 （GUIDES-28748）
- Source ビューのエレメント内にXML コメントが追加されると、ビューを切り替えると、コメントの前後のスペースが失われます。 （GUIDES-29781）
- ツールバーの省略記号アイコン（**詳細** メニュー）内にある場合、マルチメディアオプションは機能しません。 （GUIDES-29583）
- Assets UIまたはエディターを使用して新しいトピックを作成する場合、`XMLEditorConfig`で`xmleditor.uniquefilenames`設定がtrueに設定されている場合、エディターでトピックが自動的に開きません。 （GUIDES-28171）
- Source ビューでMathML数式内に追加されたスペースは、エディタービューを切り替えても保持されません。 （GUIDES-26111）
- トピックの更新中または作成中にJCR セッション接続を閉じないと、メモリリークやサービスのダウンタイムが発生します。 （GUIDES-26282）
- 列をドラッグすると、幅がパーセント値からピクセル値に変更され、テーブルが歪んだり傾いたりします（GUIDES-23128）。
- コンテンツが`code block`にペーストされた場合、または`code block`にスペースが追加され、ビューが切り替えられると、間隔が失われます。 （GUIDES-27478）
- マップを`bookmap`に追加すると、`fmditamaprefs`ではなく`fmditatopicrefs`に保存されます。 （GUIDES-25480）
- 高解像度またはズームアウトされた画面で&#x200B;**画像を挿入** ダイアログが正しくレンダリングされず、図のタイトルと代替テキストフィールドが消える。 （GUIDES-26459）
- エディターの特殊文字の挿入ボックスで、ドイツ語のロケールの読み込みに失敗する。 （GUIDES-24537）
- 新しいバージョンのDITAVAL ファイルを保存する際に追加されたコメントとラベルは、新しいバージョンのバージョン履歴に保存されません。 （GUIDES-24076）
- 右側のパネルの&#x200B;**ファイルプロパティ**&#x200B;の下にある送信および受信の参照は、マップファイルを切り替えると正しく更新されません。 この問題は、マップファイルに多数の参照が含まれている場合に特に発生します。 （GUIDES-23344）
- リポジトリーフィルターは、`ui_config.json`で定義されたカスタムフィルターの順序を保持しません。 （GUIDES-21193）
- `codeblock`要素で複数の行のテキストを削除すると、作成者ビューから削除できない空のスペースが作成されます。 （GUIDES-20672）
- **IDの自動生成** オプションが`XMLEditorConfig`で有効になっている場合でも、スニペットを使用して要素を追加したり、テンプレートを使用して作成したりすると、新しいIDが要素に対して生成されない。 （GUIDES-21734）
- DITA テンプレートから新しいDITA アセットを作成すると、対応するDITA テンプレートからレプリケーションプロパティがコピーされます。 （GUIDES-25145）
- 親フォルダー名に「&amp;」文字が含まれている場合、リポジトリーパネルからファイルプロパティにアクセスできません。 （GUIDES-25762）
- トピックファイルを閉じると、トピックがロックされている場合でも、新しいバージョンとして保存できます。 この問題は、`XMLEditorConfig`で「**閉じる**&#x200B;で新しいバージョンを求める」オプションが有効になっている場合に特に発生します。 （GUIDES-23875）
- リポジトリパネルの現在の最大幅は、コンテンツ階層が深くネストされている場合に、トピックとマップタイトルを非表示にします。 （GUIDES-27751）

## アセット管理

- Assets UIから多数のアセットを含むフォルダーをコピーすると、API タイムアウトが発生します。 操作はバックエンドで引き続き実行され、しばらくすると完了するが、UI に成功または失敗のメッセージや通知が表示されない。 （GUIDES-30900）
- Assets UIのフォルダーに対して行われたコピー&amp;ペースト操作は、後処理エラーが原因で失敗します。 （GUIDES-30840）
- Assets UIの複合アセット（サブアセットを含むアセット）を含むフォルダーで、コピー&amp;ペースト操作が失敗する。 （GUIDES-28107）
- 多数のアセットを含むフォルダーが、リポジトリで正しく読み込まれない。 （GUIDES-32500）
- フォルダーのノード名が、エディターのフォルダータイトルの代わりに誤って表示される。 （GUIDES-32402）
- ファイル名にサポートされていない文字または禁止されている文字を含むアセットをアップロードすると、エラーが発生します。 （GUIDES-28845）
- ブラウザーを更新した後に作成者ビューでトピックを開くと、ファイルのプロパティパネルで以前に適用されたタグは保持されず、新しいタグを追加すると、特に多数のタグを選択できる場合に、既存のタグが上書きされます。 （GUIDES-29078）
- 多数のアセットを含むDITA マップのメタデータレポートを生成すると、**管理** ボタンが応答しなくなり、応答が遅くなります。 （GUIDES-28443）
- トピックの作業コピーのドキュメント状態が、翻訳とベースライン UIのそのトピックのすべてのバージョンに対して表示されます。 （GUIDES-20674）

## レビュー

- レビューノードが作成されないため、AEM ワークフローを使用してレビュータスクを作成しようとすると、常に失敗します。 （GUIDES-28214）
- レビューUIのドキュメントビューでは、一部の画面サイズのコンテンツがラップされないため、コンテンツ全体を表示するには水平方向にスクロールする必要があります。 （GUIDES-25292）
- レビューダッシュボードでレビュータスクの詳細を更新しても、更新が成功したか失敗したかは確認されません。 （GUIDES-8051）

## 翻訳

- Experience Manager Guidesを通じて送信された翻訳には、添付されたアセットが含まれないため、翻訳ジョブの「**開始**」ボタンがユーザーが使用できなくなります。 （GUIDES-28237）

## 公開

- ネイティブ PDF出力では、インデックスのリスト（LOI）がアルファベット順でない順序で表示され、ネストされたインデックス用語が適切にグループ化されていないため、インデックスをナビゲートするのが困難です。 （GUIDES-29090）
- 宛先パスにコロン付きの変数が含まれている場合、AEM Sites コンポーネントマッピング出力プリセットページの&#x200B;**マップページテンプレート**&#x200B;および&#x200B;**トピックページテンプレート** ドロップダウンリストが空白で表示されます。 （GUIDES-28119）
- DITA マップにコンセプト、参照、タスクなどの様々なタイプのトピック参照が含まれており、`chunk=to-content`属性を使用して結合され、コンポーネントマッピングを使用してAEM Sitesでシングルページ出力を生成すると、公開エラーが発生してコンテンツが正しく公開されません。 （GUIDES-28118）
- `chunk=to-content`属性を持つDITA マップを公開すると、新しいAEM Sites出力に重複したJCR ノードが作成され、AEM Sitesのコンテンツ構造が冗長になります。 （GUIDES-28104）
- 新しいAEM Sites出力を使用してDITA マップを公開する際に、トピックに`chunk =to-content`属性があり、出力がトピックタイトルをページ名として使用するように設定されている場合、生成されたページ名に元のトピック名を保持する代わりに&#x200B;**chunk**&#x200B;という単語が誤って表示されます。 （GUIDES-28102）
- 新しいAEM Sites公開用のAEM Guides トピックテンプレートで設定されている`cq:template` プロパティに誤った値が表示され、テンプレート構造とコンテンツのレンダリングに影響を与えます。 （GUIDES-27789）
- DITA コンテンツに`external`のスコープを持たずにweb リンクがある場合、ネイティブ PDFの公開は無期限に続きます。 （GUIDES-26434）
- ネイティブ PDFおよびAEM サイトの公開が停止し、コンテンツにエラーがある場合にキューに入れられます。 （GUIDES-26516）
- 複数の単語をスペースで区切ったタイトルを含むAEM サイトページを生成する場合、マップタイトルにはスペースではなくハイフンが表示されます。 （GUIDES-27903）
- Native PDFの場合、無効なメタデータプロパティ名は解決されず、**ドキュメントプロパティ**&#x200B;に`unresolved property name`と表示されます。 （GUIDES-25680）
- `codeblock`内の複数行テキストは、PDF生成時にテキストオーバーフローの問題を引き起こします。 （GUIDES-15541）
- マップコレクションにマップを追加する場合、マップ以外のアセット（トピックなど）は が表示され、翻訳されたマップ言語も適切に並べ替えられません。（GUIDES-25085）
- 従来のAEM Sites出力では、マップのネストされたトピック内のセクションリンクが、Source モードで手動で設定した場合や、コンテンツが外部ソースから読み込まれた場合、正しく解決されません。 目的のセクションに移動する代わりに、ネストされたトピックを含むメイントピックにリダイレクトします。 （GUIDES-26103）
- DITA トピックの外部リンクに`scope=external`属性が見つからない場合、HTML5の公開は、この属性がエラーログに見つからないファイルを示さずに失敗します。 （GUIDES-25969）
- PDF プリセットで「**マルチメディアファイルを埋め込む**」オプションが有効になっている場合でも、ネイティブ PDFにビデオリンクを埋め込むことができません。 （GUIDES-9989）
- 新しいAEM Sites パブリッシングを使用して生成されたマッピングランディングページにメタデータプロパティを渡すことができません。 （GUIDES-27288）

## ベースライン

- Assets UIからDITA マップをコピーすると、そのアタッチされたベースラインも新しいマップにコピーされます。 （GUIDES-11227）
- 多数のラベルを持つ新しいベースラインを作成する場合、すべてのラベルが取得されるのを防ぎます。 （GUIDES-16232）

## ホームページ

- 「**最近のファイル**」ウィジェットにリストされているファイルの1つが、ソーステンプレートにサムネールが含まれていないテンプレートに基づいている場合、ホームページが空白になります。 （GUIDES-31506）

## プラットフォーム

- 大きなコレクションを操作する際に、読み込み時間の長さや断続的なタイムアウトなどのパフォーマンスの問題が発生します。 （GUIDES-29065、GUIDES-28793）
- Experience Manager GuidesにアップロードされたAEM Guides コンポーネントで使用されている非推奨のGuava ライブラリに関連する脆弱性（GUIDES-27402）

## 既知の問題

Adobeでは、5.1.0 リリースに関する次の既知の問題を特定しました。


- レビュー担当者がコメントを追加せずにタスクを完了すると、最新のタスクレベルのコメントがタスク開始者へのメール通知に表示されます。 （GUIDES-33590）
- 結合ダイアログで、選択したトピックの使用可能なバージョンを表示する代わりに、ドロップダウンリストにメインコンテンツが誤って表示されます。 （GUIDES-30820）
- 同じベースラインを使用するプリセットを切り替えると、現在のプリセットの「保存」ボタンが無効になります。 （GUIDES-28025）
- コードブロック要素内の新しい行に新しいコンテンツをペーストすると、空の行が自動的に挿入されます。（GUIDES-27842）
- DITA マップ内のトピックは、サブマップ内でkeydefとtopicrefの両方として使用されている場合、AEM Sites出力で公開できません。 （GUIDES-22269）
- コンテンツプロパティパネルで、リンクを更新ダイアログから参照を更新しようとすると、「属性」フィールドが自動的に閉じ、リンクが更新されません。 （GUIDES-17767）
