# 2026.06.0 リリースで修正された問題

この記事では、Adobe Experience Manager Guides as a Cloud Serviceの2026.06.0 リリースの様々な領域で修正されたバグについて説明します。

新機能と機能強化について詳しくは、[2026.06.0 リリース &#x200B;](whats-new-2026-06-0.md)の新機能を参照してください。

2026.06.0 リリース [&#128279;](upgrade-instructions-2026-06-0.md)の アップグレード手順について説明します。

## オーサリング

- `in`、`mm`、`px`などの単位ベースのサイズを使用して、画像プロパティダイアログの&#x200B;**幅** フィールドと&#x200B;**高さ** フィールドの間でフォーカスを切り替えると、値は安定したままではなく、徐々に増加し続けます。 （GUIDES-45929）

## エディター2.0

- `<keydef>`または`<topicref>`内の`<topicmeta>`内の`<keywords>`をコピーして貼り付けると、キーワードが不要な外部タグ内に挿入されます。 （GUIDES-45800）
- `mm`などの単位で指定した画像のサイズが正しくレンダリングされないため、指定したサイズではなく元のサイズで画像が表示されます。 （GUIDES-46275）
- タグビューを有効にしてドラッグ&amp;ドロップを実行する場合、部分的なXMLまたはDITA タグと共にコンテンツを選択すると、不要な孤立タグが残り、コンテンツまたはビューが正しく表示されません。 （GUIDES-28191）
- 表のタグビューで、折りたたまれたエントリタグのすぐ下のセルにカーソルを置いたときに上向き矢印キーを押すと、折りたたまれたタグの上をスキップして、カーソルを文書の先頭に移動します。 （GUIDES-45408）
- テーブルのコンテキストツールバーから操作を実行すると、ツールバーが予期せず閉じられ、その後のテーブル操作が中断されます。 （GUIDES-45405）
- アウトライン表示またはパンくずリストから&#x200B;**挿入後**&#x200B;または&#x200B;**挿入前**&#x200B;を使用した後、新しく追加されたタグ内ではなく、任意の位置にカーソルが移動します。 （GUIDES-45147）
- 読み取り専用モードで&#x200B;**MathML**&#x200B;を編集オプションが正しく表示されない場合、または他のユーザーがファイルをチェックアウトした場合は、ファイルを編集できないにもかかわらず、MathML コンテンツを更新できます。 （GUIDES-45172）
- バックスペースキーを使用してXML コメントを削除する場合、コメントタグはコンテンツの削除後も削除されず、削除操作は前の要素に続きます。 （GUIDES-45240）
- 画像に「**パスをコピー**」または「**UUIDをコピー**」オプションを使用すると、元のアセットパスではなく、完全なレンディションパスが返されます。 （GUIDES-45278）
- 同じトピック内のコンテンツをエディター内の無効な場所にコピー&amp;ペーストすると、意図しない`<foreign>` タグが挿入されます。 （GUIDES-45378）
- Windowsでは、テーブル行をコピーして貼り付けると、`data-pm-slice`などの予期しない属性がテーブル要素に追加され、文書の保存を妨げるマークアップエラーが発生します。 （GUIDES-45784）
- マップまたはトピックのコンテキストメニューから「**コピー**」オプションを使用し、コンテンツをペーストすると、ペーストされた出力に追加の`<data>` タグが挿入されます。 （GUIDES-45703）
- `cmd`要素から部分的な選択をドラッグする場合、コンテンツを既存のテキスト間または別の`cmd`要素の末尾にドロップすることはできません。 （GUIDES-44883）
- テーブルに`scale`属性を適用する場合、テーブルは作成者モードとプレビューモードで設定されたサイズでレンダリングされません。 （GUIDES-45984）
- ペイントやスニッピングツールなどの外部ソースからコピーした画像をペーストしても、その画像はトピックに挿入されません。 （GUIDES-45983）
- キーワードマップから派生したトピックタイトルで使用される`keyref`は、ブラウザーの更新後でも、保存後に目次またはトピックタブに表示されません。 （GUIDES-45799）
- コメント付きバージョンのサイドバイサイドビューを使用してエディターでレビュータスクを開くと、トピックに関連するバージョンではなく、バージョンが「なし」と表示されます。 （GUIDES-45127）
- レビュータスクページにアクセスする場合、トピック間のページ区切りが表示されず、コンテンツセクションが継続的にレンダリングされます。 （GUIDES-46777）
- レビューページのスタイル設定が新しいエディターのエクスペリエンスと一致しないため、一貫性のないビジュアルエクスペリエンスが得られます（GUIDES-46061）

## アセット管理

- マップにDITA以外のリソース（`.html`など）を指す外部`topicref`が含まれている場合、そのプレビューはAssets UIに表示されません。 （GUIDES-45409）
- 一括移動ツール ダイアログは、多数のソースフォルダーを選択した場合にオーバーフローし、スクロールバーが表示されないため、目的のパスフィールドとアクションボタンにマウスのみのユーザーがアクセスできなくなります。 （GUIDES-45805）
- Assets UIからアセットがコピーされると、`DXML reprocessing failure`の通知が届きます。 この通知は誤解を招くおそれがあり、コピーが正常に実行されたため、受信トレイの混乱につながります。 （GUIDES-45012）
- フォルダー内のAssetsの左側のパネルでフィルターパネルを開くと、パネルを閉じて再度開くまで、検索フィールドとファセットは無効のままになります。 （GUIDES-42652）
- 翻訳ワークフロー中に作成されたコンテンツフラグメントと中間翻訳アセットは、スケジュールされたアセット処理ジョブによって意図せずに処理されるため、ログの例外や、未処理のアセットの誤った処理が発生します。 （GUIDES-42582）

## 公開

- PDF テンプレートの`.plt`および`.css` ファイルを操作する場合、**IDを生成** オプションは、これらのファイルタイプに適用されないにもかかわらず、右クリックコンテキストメニューで使用できます。 （GUIDES-45254）
- ブラウザーの更新後に新しく作成したNative AEM Sites プリセットを再度開くと、変更が行われていない場合でも、**Save** ボタンがデフォルトで誤って有効になります。 （GUIDES-45171）
- デフォルトとしてマークされたNative PDF プリセットを複製する場合、複製もデフォルトのプリセットとして設定されます。 一度に1つのプリセットのみをデフォルトとして指定する必要があります。 （GUIDES-44786）
- `outputclass`属性は、生成されたHTMLまたはネイティブ PDF出力に反映されないため、MathML数式に適用されたカスタムクラスがレンダリング出力に影響を与えなくなります。 （GUIDES-44393）
- 新しいAEM Sites テンプレートを使用して生成されたAEM Sites出力の一括アクティベーションが失敗したため、パブリッシュインスタンスへのレプリケーションが正常に実行されませんでした。 （GUIDES-44049）
- 脆弱な`jackson-databind` JAR （バージョン 2.9.8）がDITA-OT パッケージのAEM Guidesにバンドルされていることが特定されます。 （GUIDES-43081）
- Map Collectionからネイティブ AEM Sites出力を開くと、リソースが見つからないというエラーが発生し、生成された出力へのアクセスが妨げられます。 （GUIDES-42065）
- ネイティブ PDFの生成中に、DITA マップの`<reltable>`要素が無視され、「関連コンセプト」、「関連タスク」、および類似の自動生成された相互参照セクションが出力に表示されなくなります。 （GUIDES-38333）
- AEM Sites上の`processing-role=resource-only`要素を含むDITA マップを公開する場合（従来のコンポーネントマッピングを含む）、孤立したサイトページは、`topicgroup`要素や特定のコンテンツ設定などの追加のシナリオでそれらの要素に対して生成されます。 （GUIDES-37650）
- 長い名前のマップがマップコレクションに追加されると、マップコレクション UIは歪んだレイアウトでレンダリングされます。 この問題は、新しいマップ コレクションで解決されました。 （GUIDES-42062）
- Native PDF engine v1での公開：
   - 特定のコンテンツに対してPDFのネイティブ出力を生成する場合、複数のページにまたがるコンテンツ全体が含まれる中間HTMLにもかかわらず、PDFでは最初のページのみがレンダリングされます。 （GUIDES-28270）
   - アクセシビリティ設定が有効になっているNative PDF出力のコンテンツの読み取り順序が正しくありません。 フッターのページ番号は、最後ではなくメインコンテンツの前に読み取られます。 （GUIDES-27790）
   - ネイティブPDF出力のカラーバーは、ページ全体の幅に広がらず、ページサイズをカスタマイズすると重なり合い、一部のカラーボックスが非表示になります。 （GUIDES-15505）
   - CSS `:is()`の擬似クラス セレクターは、ネイティブ PDF出力で尊重されないため、ブラウザーのレンダリングと比較してスタイルの違いが生じます。 （GUIDES-11328）

  >[!NOTE]
  >
  > 上記の問題は、Native PDF engine v2で解決されました。 詳しくは、[&#x200B; ネイティブ PDF エンジン v2](../native-pdf/new-pdf-engine.md)の操作を参照してください。

## 翻訳

- 多くのアセットを含むマップにベースラインを適用すると、選択した言語の翻訳レポートの読み込みが遅くなり、レポートのレンダリング前にタイムアウトがリクエストされることがあります。 （GUIDES-45508）
- 選択した言語フォルダーがアンダースコア形式（`da_dk`）ではなくハイフネーション付きのロケールコード（`da-DK`など）を使用している場合、「Workspace設定」の「翻訳」タブの言語グループは保持されません。 （GUIDES-37843）

## レビュー

- トピック名の横にあるレビューUIの左側のパネルに、**バージョン履歴** アイコンのツールヒントがありません。 （GUIDES-45511）
- アクティブなレビュータスクを編集する際に、既にレビューの一部となっているトピックが削除され、**更新**&#x200B;を選択せずに再追加された場合、そのトピックのレビュー担当者情報は失われます。 （GUIDES-38774）
- レビューUIのコンテンツペインには、推奨される1600 pxの解像度など、特定の画面解像度に対して水平スクロールバーが表示されます。これにより、コンテンツがペイン幅を超えると、レビューコメントが非表示になります。 （GUIDES-44082）

## 学習コンテンツ

- 「質問バンクから挿入」オプションを使用してクイズに質問を追加する場合、有効な質問IDがあるにもかかわらず、簡潔な回答の質問はリストされません。 （GUIDES-44942）
- 「質問バンクから挿入」オプションを使用して質問を挿入すると、質問バンクから挿入ダイアログが開くときに予期せず点滅したり、サイズが変更されたりします。 （GUIDES-45524）
- 「質問バンクから挿入」オプションを使用して質問を挿入すると、質問タイトルに画像が含まれている場合にエラーが発生します。 （GUIDES-45383）
- SCORM出力テンプレートを選択すると、テンプレートタイトルが変更されたときにアプリケーションエラーが発生します。 （GUIDES-45521）
- サブタイトル （`.vtt`）ファイルは、フォルダーにアップロードされた後、リポジトリまたはエクスプローラーのビューに表示されません。 （GUIDES-46014）
- HTML プレビューでビデオコンテンツを読み込む際、ソース （`src`）のパスには、元のファイルパスを保持する代わりに追加のレンディションパスが含まれるため、ビデオが正しくレンダリングされません。 （GUIDES-41776）
- Safari ブラウザーからSCORM出力を公開すると、パッケージがzip ファイルではなくフォルダーとしてダウンロードされ、生成されたコンテンツのレンダリングと機能の問題（拡張コンテンツ、非動作アコーディオンなど）が発生します。 （GUIDES-45119）
- 「次へ」ボタンがコースに対して有効になるまでに遅延が発生し、学習者のナビゲーションエクスペリエンスに影響が及びます。 （GUIDES-45113）
- プレビューで正しく表示されているにもかかわらず、出力を公開する際に、簡潔な質問のスタイルが正しくレンダリングされない。 （GUIDES-46478）
- ビデオを含むコースのPDF出力を生成する場合、ビデオコンテンツはレンダリングされず、ビデオサムネールやポスター画像などのプレースホルダー画像の代わりにファイルパスのみが表示されます。 この問題は、ネイティブ PDF出力プリセットで「**マルチメディアファイルを埋め込む**」オプションを有効にすることで解決されました（GUIDES-45117）。
- CSSの疑似クラスと要素は、ガイドのエディターで白でレンダリングされるので、エディターの背景に対して見えないか読みづらくなります。 （GUIDES-45116）

## 既知の問題

Adobeでは、2026.06.0 リリースの次の既知の問題を特定しました。

## エディター2.0

- Sourceとオーサーモードを切り替えると、コンテンツの不整合が発生し、トピックの一部が消えたり、モード間で反映されなくなったりします。 （GUIDES-47432）

- **変更履歴**&#x200B;を有効にしてアウトライン ビューで作業する場合、変更を拒否すると、特定の変更されたコンテンツのみを拒否するのではなく、タグ内のコンテンツ全体が削除されます。 （GUIDES-48319）

- プレビューモードの&#x200B;**PDFとして書き出し** ボタンは、フォルダープロファイルで`editor_toolbar.json`を使用してエディターツールバーをカスタマイズする場合、何の操作も実行しません。 （GUIDES-47525）

- 削除操作を実行する際に、画像マップ、構造化要素、インライン書式設定タグ、およびマージできないブロックで、カーソルの移動やナビゲーションに多少の不整合が発生する可能性があり、予期しないカーソルや削除動作が発生することがあります。 （GUIDES-46756）

- マップエディターで壊れたリンクで`Ctrl+click`を使用すると、アプリケーションエラーがトリガーします。 （GUIDES-45544）

- 読み取り専用コンテンツ（`conref`段落など）の直後の段落の先頭でバックスペースを押すと、編集可能な段落が予期せず削除または結合され、編集可能な段落が予期せず削除される可能性があります。 （GUIDES-45049）

- `bodydiv`などの要素に条件インジケーターを適用すると、フルタグビューの隣接するタグにインジケーターがオーバーフローし、誤った視覚的なレンダリングが発生します。 （GUIDES-44971）

- エディターで作業している場合、`keyrefs`またはマップから派生したキー（`term`または`abbreviated-form`要素で参照されているマップ内の用語集エントリを含む）を選択できず、オーサリングエクスペリエンスが低下します。 （GUIDES-45790） <br> **回避策**：属性値を使用して、右側のパネルから`keyref`の値を変更できます。

- アウトライン表示で、トピックを閉じて再度開いた後に&#x200B;**テキストを表示**&#x200B;を再度有効にしても、エレメントタグと一緒にテキストが表示されません。 （GUIDES-48320） <br> **回避策**:「**テキストを表示**」オプションを有効にしてから、トピックを再度開きます。 再度開くと、テキストがエレメントタグの横に正しく表示されます。

- インラインタグの名前を&#x200B;**要素の名前を変更** オプションを使用して変更すると、パンくずリストはすぐに更新されず、カーソルをタグに移動するかビューモードを変更した後にのみ変更が反映されます。 （GUIDES-44993） <br> **回避策**: インラインタグの名前を変更してブラウザーを更新し、パンくずリストを更新します。

- MathML数式を`conref`として挿入すると、正しくレンダリングされません。 （GUIDES-46601） <br> **回避策**: MathML数式を`<p>`要素の中にラップし、その`<p>`要素をconref ソースとして使用します。

## レビュー

- レビュータスクがプロジェクトチーム外のユーザーに再割り当てされると、レビューアーの表示、レビューステータスの追跡、委任通知が正しく機能しないにもかかわらず、ユーザーはプロジェクトメンバーシップを欠いてもレビューアクションを実行できます。 （GUIDES-46602）

## 公開

- 条件プロファイリングを使用してすべての箇条書きリスト項目を除外するDITAVAL フィルタリングを使用してコンテンツを公開する場合、出力はリスト構造全体を削除する代わりに、空の箇条書きマーカーを誤って保持します。 （GUIDES-47238）

- 新しいベースラインでネイティブ AEM サイト出力を公開すると、トピックリストが空白になり、選択したベースラインに含まれるトピックが表示されません。 （GUIDES-46480） <br> **回避策**：古いベースラインを使用して、ネイティブ AEM サイト出力を公開します。これは、Configuration Managerで設定できます。
