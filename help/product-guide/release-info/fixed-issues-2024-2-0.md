# 2024.2.0 リリースで修正された問題

この記事では、Adobe Experience Manager Guides as a Cloud Serviceの2024.2.0 リリースの様々な領域で修正されたバグについて説明します。

新機能と機能強化について詳しくは、[ 2024.2.0リリースの新機能](whats-new-2024-2-0.md)を参照してください。

2024.2.0 リリース ](upgrade-instructions-2024-2-0.md)の[ アップグレード手順について説明します。



## オーサリング

- エディターのスペルチェックでは、候補を選択できません。 (15045)
- グローバルナビゲーションボタンが機能せず、ダッシュボードの読み込みに失敗する。 (14968)
- Web エディターで、ダウンロードマップ機能を使用すると、ダウンロードの準備ができたときにポップアップ通知がトリガーされない。 (14626)
- Web エディターで、マップのダウンロード機能がベースライン付きのマップのダウンロードに失敗します。 (14622)
- Experience Manager Guides as a Cloud Serviceの2023年10月リリースのDTD エラーが無効です。 (14482)
- リポジトリから用語集トピックを用語集マップにドラッグすると、`topicref`が作成されます。 (10767)
- スニペットのプレビュー画面がフリーズする。 (14840)
- `labels.json` ファイルのラベルは、Web エディターでランダムに表示されます。 (10508)

## 公開

- ネイティブ PDF パブリッシングでは、条件プリセット内のカスタム属性は、ネイティブ PDF パブリッシングでは機能しません。 (14943)
- ネイティブ PDF パブリッシングでは、Adobe Experience Manager Guidesの2023年12月リリースの主要な参照が解決されません。 (15085)
- AEM Sitesの公開が失敗し、「HTTP」で始まるDITA ファイルに`xref`を持つファイルのスコープエラーが発生します。 (15154)
- エディターの「**出力**」タブからカスタムテンプレートを追加できません。 (14846)
- 空のテンプレートパスが原因で、**AEM サイト** プリセットが機能しません。 (14804)
- AEM数式を含むトピックを含むDITA マップで、MathML サイトの再生成が失敗します。 (14790)
- ネイティブ PDF パブリッシングでは、PDF生成で`Node.js`公開用の依存関係を取得する際にエラーが発生します。 (14445)
- AEM プリセットでは、Web エディターで`/content`階層外のテンプレートを選択することはできません。 (14260)
- AEM Sites出力では、サブエレメントを持つ`<lines>` エレメントのスタイルまたは改行が失われました。 (12542)
- カスタムメタデータは、最終出力では使用できません。 (12116)
- ネイティブ PDF パブリッシングでは、DITA マップメタデータプロパティを使用してPDF ファイル出力用のメタデータを入力することはできません。 (15159)



## 管理

- **ベースラインフィルター**&#x200B;のファイルが、Web エディターのファイル名で機能しません。 (13486)
- パフォーマンスを向上させるために親DITA マップのインデックス作成を無効にすると、特定の機能に影響を与える可能性があります。（12213）


## レビュー

- **Accept**&#x200B;または&#x200B;**Reject**&#x200B;の変更履歴では、右クリックコンテキストメニューが機能しません。 (14607)
- Adobe Experience Manager Guidesの2023年12月リリースで、レビュー画面でDITA トピックを閉じるには切り替えできません。 (14537)
- レビューワークフロー用にメールテンプレートをカスタマイズすると、オーバーレイでは機能しません。 (13954)

## 既知の問題

Adobeでは、2024.2.0 リリースの既知の問題を次のように特定しました。

- 複製されたDITA ファイルのUIにバージョン 1.0は表示されません。
- 2024.2.0 リリースでは、どのプリセットでもマイクロサービスを有効にすると、アセットメタデータの伝達が機能しません。
