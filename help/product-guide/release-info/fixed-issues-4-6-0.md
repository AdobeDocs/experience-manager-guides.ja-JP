# 4.6.0 リリース（2024年9月）で修正された問題


この記事では、Adobe Experience Manager Guidesの4.6.0 リリースの様々な領域で修正されたバグについて説明します。


新機能と機能強化について詳しくは、[4.6.0 リリースの新機能](whats-new-4-6.md)を参照してください。

4.6.0 リリース ](../release-info/upgrade-instructions-4-6-0.md)の[ アップグレード手順について説明します。

- **翻訳ステータス**&#x200B;が&#x200B;**進行中**&#x200B;の翻訳プロジェクトを選択すると、間違ったページが開きます。 (13248)
- 「**構造のみを作成**」オプションを選択して作成した翻訳プロジェクトには、UUIDが割り当てられていません。 (18980)


## 既知の問題

Adobeでは、4.6.0 リリースに関する次の既知の問題を特定しました。
- **AEM Sites** プリセットを開くと、トピックがダーティとしてマークされます。
- 選択したパネルは、**出力** タブからブラウザーの更新時に保持されません。
- **作成者** ビューの2つの`topicrefs`間でトピックをドラッグ&amp;ドロップできません。
- プリセットに適用された条件フィルタリングが、**PDFとしてダウンロード**&#x200B;経由で適用されません。
- マップパネルから1つのトピックを生成すると、**AEM Sites** プリセットで選択されたすべてのトピックが生成されます（従来のトピック以外）。
- DITA マップの上部ツールバーから挿入すると、トピックの参照がユーザーインターフェイスで壊れて表示されます。
- DITA マップに参照がない場合、ネイティブ PDFの生成が失敗します。
- トピックのドキュメントの状態が&#x200B;**完了**&#x200B;に更新されると、**新しいリリースの開始** アイコンは、トピックの&#x200B;**プレビュー** モードでのみ使用できます。
- Asset UIでDITA ファイルを選択すると、コンフィギュレーション設定で無効になっていても、「**FrameMakerで開く**」オプションが表示されます。
