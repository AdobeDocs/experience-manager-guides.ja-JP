# 2024.06.0 リリースで修正された問題

この記事では、Adobe Experience Manager Guides as a Cloud Serviceの2024.06.0 リリースの様々な領域で修正されたバグについて説明します。

新機能と機能強化について詳しくは、[&#x200B; 2024.06.0リリースの新機能](whats-new-2024-06-0.md)を参照してください。

2024.06.0 リリース [&#128279;](upgrade-instructions-2024-06-0.md)の アップグレード手順について説明します。

## オーサリング

- 15 KBを超えるトピックのコピー&amp;ペースト操作は、予期しないエラーで失敗します。 (17171)
- **ファイルのプロパティ** パネルからドキュメントの状態を変更する機能が正しく機能せず、*ドラフト*&#x200B;状態に変更されます。 (17088)
- カスタムフォルダープロファイルを使用してXML エディターの設定を変更すると、`ui_config.json`が正しくないファイルで更新されます。 (17011)
- **リポジトリ** パネルで、**詳細フィルター** ダイアログボックスを再度開くときに&#x200B;**フィルター**&#x200B;検索で&#x200B;**チェックアウト済み** フィールドの値が保持されない。 (16935)
- トピックからのリンクされた画像は、バージョン作成後にベースラインに表示されません。 (16931)
- **ユーザー設定**&#x200B;が&#x200B;**ファイル名**&#x200B;でファイルを表示するように設定されている場合、再利用可能なコンテンツパネルに要素が一覧表示されません。 (16896)
- テーブルタイトル要素内のサブエレメントは、Experience Manager Guidesの&#x200B;**プレビュー** モードでレンダリングできません。 (16691)
- **FileNotFoundException**&#x200B;の例外が原因で、postprocess スクリプトの実行が失敗しました。 (16517)
- Vimeoの動画は、Experience Manager Guidesのフルスクリーン機能をサポートしていません。 (15996)
- `<pre>`または`<codeblock>`要素に長い書式が設定されたテキストシーケンスをペーストすると、テキストが切り捨てられます。 (15859)
- コンテンツの削除は、テンプレートがコードを介してインストールされても未処理のままである場合に、重複するGUIDが原因で発生します。 (15858)
- Experience Manager Guidesは、**プレビュー** モードで&#x200B;**処理役割**&#x200B;属性に準拠できません。 (15787)
- エディターは、選択した領域を超えて余分なテキストを断続的に削除します。  (15708)
- Word ドキュメントまたはHTMLからWeb エディターに大きなテーブルをコピーして貼り付けることができない。 (15369)
- Experience Manager Guides as a Cloud Serviceの空のフォルダーに対して&#x200B;**Copy**&#x200B;関数が失敗します。 (15353)
- 要素への属性の追加や新しい要素の挿入をキャプチャするAPIやイベントがない。 (15351)
- Web エディターで`<ol>` タグ内の`<data>` タグを追加できません。 (15161)
- 統合シェルが有効になっている場合、ラベルのないバージョンの&#x200B;**バージョンラベル管理** ダイアログボックスに&#x200B;**メインコンテンツ**&#x200B;が誤って表示されます。 (15039)
- web エディターでは、大きなファイルの読み込みが数秒遅くなり、読み込みが遅くなります。 (14958)
- テキスト内の表セルで&#x200B;**Enter** キーを押しても、段落が期待どおりに分割されません。 (14251)
- 自動インデント機能を使用した後、Experience Manager DITA ガイドで&#x200B;**Save**&#x200B;関数のトリガーに失敗します。 (16482)
- レビュートピックが正しい順序で表示されない。 (16319)
- **作成者** ビューでは、改行しないスペースを使用するとコピー&amp;ペーストの問題が発生し、テキストがオーバーフローします。 (15541)

- `<topicmeta>`内の`<othermeta>`要素では、`<conref>`を別のDITA マップに追加すると、マップ IDも要素のIDと共に追加されます。 (15226)
- 変更を行うと、`<conref>`を&#x200B;**属性** パネルから更新できません。 (15209)
- 表セル内の画像を選択すると、セル全体が選択されます。 (15188)

## 公開


- クロスマップリンクは、`peer @scope`を持つリンクの公開コンテキスト設定のすべての親マップを表示できません。 (16700)
- 新しい属性を追加するか、既存の属性を削除すると、古い属性は&#x200B;**条件プリセット**&#x200B;に保持されます。 (15890)
- ネイティブ PDF パブリッシング出力で、RTL言語コンテンツが正しく処理されない。 (15709)
- 最初のPDFは、ネイティブPDF出力が生成されたときにバージョン管理されません。 (10305)
- ネイティブ PDFでは、ネストされたDITA トピックが目次（TOC）に正しく表示されません。 (16742)
- ビデオファイル用にDynamic Mediaから生成されたサムネールは、公開された出力に保持されません。 (15656)
- ARM64 プロセッサーでのネイティブ PDF パブリッシングの出力生成が失敗します。 (16968)

## 管理

- 既存のベースラインを編集し、特定のバージョンを選択すると、アプリケーションエラーがトリガーされます。 (14451)

## 翻訳

- Experience Manager Guidesの2023年10月リリースでは、Adobe Experience Manager 6.5 SP18に新しい言語ジョブを追加できません。 (15398)

## Cloud Service

- Adobe Experience Manager ツールのナビゲーションが応答しません。 (17118)
- Cloud Services デプロイメントのビルド変換フェーズが、DITA コードベースのエラーで失敗します。 (14432)

## レポート

- DITA アセットをコピーする際にパッチが適用されていないため、Experience Manager Guides UIで不正確な&#x200B;**トピックリスト**&#x200B;がカウントされ、DTIA マップ用に生成されたレポートに影響を与えます。 (15529)
- URLの&#x200B;*%20*&#x200B;の外部参照を含むトピックには、壊れたファイル参照が表示されます。 (15347)


## 既知の問題

Adobeでは、2024.06.0 リリースの次の既知の問題を特定しました。

- Vimeo コンテンツがトピックに追加されると、ネイティブ PDFの公開が失敗します。
- **トピックプロパティ**&#x200B;は、ページレイアウトのメタデータフィールドで選択した形式に従って表示されません。
- Dynamic Mediaが有効になっている場合、`xrefs`は&#x200B;**Assets** ビューでクリックできません。
