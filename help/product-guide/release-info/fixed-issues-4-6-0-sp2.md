# 4.6.0 サービスパック 3 リリース（2025年1月）で修正された問題


この記事では、Adobe Experience Manager Guidesの4.6.0 Service Pack 3 リリースの様々な領域で修正されたバグについて説明します。

4.6.0 サービスパック 3 リリース ](upgrade-instructions-4-6-0-sp2.md)の[ アップグレード手順について説明します。

## オーサリング

- フォルダープロファイルから条件を更新すると、すべての条件グループが失われ、条件が統合されます。 (23526)
- **コンテンツプロパティ** パネルのテーブルのヘッダー行の値を変更しても、更新された値は適用されません。 (23213)
- レイアウトビューで&#x200B;**トピック参照**&#x200B;要素ダイアログを使用してDITA マップに新しいトピック参照を追加すると、追加された参照が壊れたリンクとして表示されます。 (22859)
- 作成者ビューで&#x200B;**トピック参照**&#x200B;要素ダイアログを使用してDITA マップにトピックを追加すると、選択したトピックが選択された順に挿入されます。 (22858)
- 外部製品（MS PowerPointなど）から画像をコピーしてGuidesに貼り付けると、ファイルで使用するためにアセットを即座にアップロードする機能が壊れます。 (24983)
- **検索とフィルター**&#x200B;機能を使用してリポジトリ内のファイルを検索する場合、複数のファイルを選択できません。 (25104)

## 公開

- Salesforceへの公開は、コンテンツに非改行スペースが含まれている場合に失敗します。 (23664)
- リンク切れなどのエラーがあるトピックの場合、Salesforceの公開は失敗し、進行状況バーは無期限に表示されます。 (22985)
- リンク切れのマップの場合、Salesforceの公開が失敗し、進行状況バーが無期限に表示されます。 (24963)
- 外部リンクにUUIDが含まれている場合、そのリンクは後処理に入り、外部リンクをUUID リンクに変換します。これにより、web エディターおよび公開サイト上のリンクが壊れます。 (22574)
- リンクの&#x200B;**スコープ**&#x200B;が&#x200B;**外部**&#x200B;に設定されている場合でも、`xref`は相対リンクに変換されます。 (23059)
- **chunk**&#x200B;属性が&#x200B;**to-content**&#x200B;に設定されているコンテンツに対して、ネイティブ PDFの生成が失敗します。 (21772)
- ベースラインの&#x200B;**プロパティを編集** ダイアログに、以前に保存した動的ベースラインの基準が表示されません。 (23964)


## 翻訳

- 非レガシー翻訳（非UUID）の場合、ソースパス内のトピックを別のフォルダーに移動すると、ターゲットロケールで参照が壊れます。 (24152)
