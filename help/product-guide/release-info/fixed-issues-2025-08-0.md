# 2025.08.0 リリースで修正された問題

この記事では、Adobe Experience Manager Guides as a Cloud Serviceの2025.08.0 リリースの様々な領域で修正されたバグについて説明します。

新機能と機能強化について詳しくは、[&#x200B; 2025.08.0リリースの新機能](whats-new-2025-08-0.md)を参照してください。

2025.08.0 リリース [&#128279;](upgrade-instructions-2025-08-0.md)の アップグレード手順について説明します。

## オーサリング

- ネイティブ PDF テンプレートの&#x200B;**CSS**&#x200B;および&#x200B;**ページレイアウト**&#x200B;のファイルでは、ファイルのロック動作に一貫性がなく、ファイルがロックされている場合でも編集が可能です。 （GUIDES-26688）
- 左側のパネルにカスタムボタンを追加した後にページを更新すると、重複したタブが作成されます。 （GUIDES-30899）
- トピックの`code block`要素内に角括弧（&lt;/または/>など）を含むXML コンテンツを追加すると、最終的な出力でコードブロック要素が空白で表示されます。 （GUIDES-31007）
- ファイルをエディターツールバーからチェックアウトしてチェックインすると、グローバル変数`canCheckIn`および`canCheckOut`の値が正しく設定されません（GUIDES-29890）。
- 統合シェルを有効にしてDITA マップを開くと、エディターが断続的に更新されます。（GUIDES-26919）
- マップビューでDITA マップを選択すると、マップの子ノードが選択されていないため、最初に選択回数が正しく表示されません。 正しい選択数とすべての子ノードの包含は、その後の選択でのみ反映されます。 （GUIDES-25663）
- **DITA トピック** フィルターを使用してリポジトリーパネルで検索しても、マークダウンファイルは取得されません。 また、**検索と置換**&#x200B;は、Markdown ファイルおよび関連コンテンツでは機能しません。 （GUIDES-27133）
- 拡張機能フレームワークを使用して、エディターツールバーの&#x200B;**メニュードロップダウン**&#x200B;をカスタマイズできません。 その結果、既存のボタンを非表示または表示したり、メニュードロップダウンに新しいボタンを追加したりすることはできません。 （GUIDES-28748）

## アセット管理

- Assets UIから多数のアセットを含むフォルダーをコピーすると、API タイムアウトが発生します。 操作はバックエンドで引き続き実行され、しばらくすると完了するが、UI に成功または失敗のメッセージや通知が表示されない。 （GUIDES-30900）
- Assets UIのフォルダーに対して行われたコピー&amp;ペースト操作は、後処理エラーが原因で失敗します。 （GUIDES-30840）
- Assets UIの複合アセット（サブアセットを含むアセット）を含むフォルダーで、コピー&amp;ペースト操作が失敗する。 (28107)
- 多数のアセットを含むフォルダーが、リポジトリで正しく読み込まれない。 （GUIDES-32500）
- フォルダーのノード名が、エディターのフォルダータイトルの代わりに誤って表示される。 （GUIDES-32402）
- ファイル名にサポートされていない文字または禁止されている文字を含むアセットをアップロードすると、エラーが発生します。 （GUIDES-28845）

## 公開

- ネイティブ PDF出力では、インデックスのリスト（LOI）がアルファベット順でない順序で表示され、ネストされたインデックス用語が適切にグループ化されていないため、インデックスをナビゲートするのが困難です。 （GUIDES-29090）
- 宛先パスにコロン付きの変数が含まれている場合、AEM Sites コンポーネントマッピング出力プリセットページの&#x200B;**マップページテンプレート**&#x200B;および&#x200B;**トピックページテンプレート** ドロップダウンリストが空白で表示されます。 （GUIDES-28119）
- DITA マップにコンセプト、参照、タスクなどの様々なタイプのトピック参照が含まれており、`chunk=to-content`属性を使用して結合され、コンポーネントマッピングを使用してAEM Sitesでシングルページ出力を生成すると、公開エラーが発生してコンテンツが正しく公開されません。 （GUIDES-28118）

## ベースライン

- Assets UIからDITA マップをコピーすると、そのアタッチされたベースラインも新しいマップにコピーされます。 （GUIDES-11227）

## ホームページ

- 「**最近のファイル**」ウィジェットにリストされているファイルの1つが、ソーステンプレートにサムネールが含まれていないテンプレートに基づいている場合、ホームページが空白になります。 （GUIDES-31506）

## 既知の問題

Adobeでは、2025.08.0 リリースの次の既知の問題を特定しました。

- エディターで開いたファイルの名前を変更または移動すると、モード（**作成者**、**プレビュー**&#x200B;など）を切り替えると、編集領域のコンテンツが更新されますが、右下隅のアクティブモードは視覚的にハイライト表示されません。 （GUIDES-32719） <br> **回避策**：問題を解決するには、ページを更新してください。
- ファイル名にスペースが含まれる画像は、条件付き属性を使用してフラグ付けすると、出力に表示されません。 （GUIDES-33858）
- **コンテンツのプロパティ** パネルで、**リンクを更新** ダイアログから参照を更新しようとすると、「属性」フィールドが自動的に閉じ、リンクが更新されません。 （GUIDES-17767）
