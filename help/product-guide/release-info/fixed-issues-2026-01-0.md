# 2026.01.0 リリースで修正された問題

この記事では、Adobe Experience Manager Guides as a Cloud Serviceの 2026.01.0 リリースの様々な領域で修正されたバグについて説明します。

新機能と機能強化について詳しくは、[&#x200B; 2026.01.0リリースの新機能](whats-new-2026-01-0.md)を参照してください。

[2026.01.0 リリースのアップグレード手順 &#x200B;](upgrade-instructions-2026-01-0.md) について説明します。

## オーサリング

- コンテキストメニューの「MathMLを編集」オプションを使用してインライン MathMLの式を更新する場合、更新された値は、ページが更新されるまで反映されません。 （GUIDES-38198）
- トピックに、再利用可能な要素（ID を持つ要素）が再利用可能なパネルに多数含まれている場合、コンテナの高さが固定されているので、一部の要素にアクセスできないことがあります。 （GUIDES-37220）
- ファイルにクロスリファレンスを挿入する場合、マップとトピックのアイコンは同じです。（GUIDES-36662）
- マップの編集時に、オーサービューでは `navtitle` の特殊な記号が `topichead` に表示されません。 （GUIDES-35435）
- **新しいバージョンとして保存** ダイアログからトピックに複数のバージョンラベルを追加できない。 （GUIDES-32716）

## アセット管理

- Assets UI のバージョン履歴パネルからバージョンラベルを削除できない。 （GUIDES-38276）
- ファイル名に無効な文字が含まれるアセットをアップロードすると、アセットのロックが解除されているにもかかわらず、そのアセットのアップロードに失敗し、誤ったメッセージが表示されます **他のユーザーによってファイルがロックされています**。 （GUIDES-32680）
- Assets検索で、サブアセットとメタデータノード（画像や PDF など）が結果に誤って含まれます。 さらに、DITAVAL フィルターが適用された場合の出力プリセットについては、条件プリセットから作成された内部的に生成された DITAVAL ファイルが検索で返されます。 （GUIDES-35418）

## 公開

- AEM Sites出力を生成する際に、キーワードを含むマップタイトルと `<ph>` の要素を含むトピックタイトルが、公開された出力に含まれません。 （GUIDES-36641）
- 範囲が外部として設定されていない PDF へのリンクを含むコンテンツを含むカスタムプリセットを使用して公開すると、プロセスが完了しません。 （GUIDES-21708）
- 一括有効化を実行すると、パッケージの作成によって、ページの `fileReference` プロパティの下にリストされているすべてのパスに対するフィルターが追加されます。これには、外部パスとピアパスが含まれます。 （GUIDES-24887）
- Native PDF出力では、`abbreviated-form` 要素は、指定された `glossterm` または `glossSurfaceForm` の代わりに `glossAcronym` を表示します。 （GUIDES-26393）
- ネイティブ PDF出力の場合、画像の `<alt>` 要素は無視され、アクセシビリティのために代替テキストが適用されなくなります。 （GUIDES-29087）
- プリセットの公開中にベースラインを含むマップの一時ファイルをダウンロードすると、`metadata.xml` ファイルは `versionPath` ではなく `dampath` を誤って参照します。（GUIDES-29815）
- 引用を含むトピックを作成または編集する際に、引用ダイアログに作成者フィールドが追加されていない場合、PDFが生成されません。 （GUIDES-37934）
- CSS が参照されていないにも関わらず、CSS ファイル（`rhdefault.css`）がPDF テンプレートに誤って適用され、CSS ファイルのエラーログが見つからない。（GUIDES-31752）

## プラットフォーム

- トピックまたはマップを保存しようとすると、特にバックグラウンドで実行されている集中的なアセット処理タスクまたは翻訳ワークフロー中に、**ファイルの保存に失敗しました** エラーが発生して、操作が断続的に失敗することがあります。 （GUIDES-37837）
- トピックまたはマップ内の DAM コンテンツへの参照に `scope="external"` を使用すると、アセットの相対パスが GUID に置き換えられます。 （GUIDES-38783）

## レポート

- 壊れたリスト レポートには、現在のマップの範囲内で正しく解決された外部リンク、有効な `keyrefs`、およびキーワードが誤って含まれています。 （GUIDES-27774）

## 既知の問題

Adobeでは、2026.01.0 リリースの次の既知の問題を特定しました。

- レビュー中のトピックが進行中のレビュータスクから削除された場合、そのトピックがレビュータスクの一部でなくなった場合でも、ドキュメントの状態は引き続き **レビュー中** となります。 （GUIDES-38709） <br>**回避策**：トピックのドキュメントの状態を **レビュー中** から「プロパティ」ページまたは「ファイル」プロパティパネルで適切な状態に変更します。
- **検索と置換** を使用して検索を実行する場合、検索結果からファイルを開いてから閉じてから、リストされた結果を選択して再度開こうとすると、ファイルを再度開くことができない。 （GUIDES-39050） <br>**回避策**：検索結果から他のファイルを開いてから、以前に閉じられたファイルをリストから再度開いて問題を解決します。
- DB サーバーでガイドを使用する場合、自己参照を含むコンテンツに対して、トピックリストレポートで各自己参照に対して無効なエントリが表示され、ファイル数が不正確になります。 （GUIDES-39420）
