# コースの公開について

コースを公開することは、対象オーディエンスが学習コンテンツにアクセスできるようにする最後のステップです。 現在、コースは、提供方法に応じて、SCORM （共有可能なコンテンツオブジェクト参照モデル）形式またはPDF形式で公開できます。
