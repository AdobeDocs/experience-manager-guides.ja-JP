# コースの公開について

コースの公開は、学習コンテンツを目的のオーディエンスが利用できるようにするための最後のステップです。 現在、コースの配信方法に応じて、SCORM （共有可能コンテンツオブジェクト参照モデル）またはPDF形式でコースを公開できます。

ここでは、SCORMおよびPDF出力を作成、設定、生成する方法を視覚化するのに役立つ簡単なビデオの概要を示します。
