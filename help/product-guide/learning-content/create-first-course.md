# 最初のコースを作成する

Experience Manager Guidesのコースは、さまざまな学習目標に合わせて設計できます。 通常の学習コースには、トピック、クイズ、要約が含まれますが、主に評価に焦点を当てたコースを構築することもできます。 例えば、クイズのみのコースや、クイズと1つの概要トピックを設定して、理解していることを素早く確認できます。 また、事前評価クイズ、メインコースコンテンツ、最終クイズを利用して、構造化されたパスを構築することもできます。 これらのオプションは、学習者の進捗状況を効果的に測定しながら、ターゲットを絞った学習体験を提供するのに役立ちます。

ステップバイステップのプロセスに入る前に、最初のコースを作成してコンポーネントを追加する方法を示す簡単なチュートリアル動画を用意しています。

>[!VIDEO](https://video.tv.adobe.com/v/3469537/aem-guides-learning-content?quality=12&learn=on)


最初のコースを作成するには、次の手順を実行します。

1. コースを作成するフォルダーに移動し、**オプション** メニューから&#x200B;**新規/ コース**を選択します。
   ![](assets/create-new-course.png)

   **新しいコースダイアログ**&#x200B;が表示されます。
2. **新規コースダイアログ**&#x200B;で、次の詳細を入力します。
   - コースのベースとなるテンプレート。

     >[!NOTE]
     >
     > 管理者が設定したコーステンプレートのみを表示します。

   - コースのタイトル。
   - コースのファイル名。 ファイル名は、コースタイトルに基づいて自動的に提案されます。 管理者がUUID設定に基づいて自動ファイル名を有効にしている場合、ファイル名フィールドは表示されません。
   - コースを保存するパス。 デフォルトでは、リポジトリで現在選択されているフォルダーのパスが「パス」フィールドに表示されます。
3. 「**作成**」を選択します。
選択したテンプレートに基づいて、指定したパスでコースが作成されます。 また、コースはコースマネージャーで開いて編集できます。

   ![](assets/course-manager-read-only-mode.png)
