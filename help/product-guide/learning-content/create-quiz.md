# クイズの作成

コースにクイズを追加するには、次の手順を実行します。

1. **コースマネージャー**&#x200B;でコースを開き、**オプション** メニューから&#x200B;**新しい**&#x200B;を追加を選択します。

   ![](assets/workflow-quiz.png){width="650"}

1. 「**クイズ**」を選択します。\
   **新しい学習クイズ** ダイアログが開き、クイズの関連する詳細を指定します。 ドロップダウンメニューからテンプレートを選択し、そのテンプレートに適したタイトルを指定できます。

   ![](assets/new-learning-quiz.png){width="350"}

1. 「**作成**」を選択します。

クイズがコースの一部として追加され、コースマネージャーパネルに表示されます。

>[!NOTE]
>
>  クイズを作成すると、自動的にバージョン 1.0が割り当てられます。
