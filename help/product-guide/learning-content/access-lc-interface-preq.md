# インターフェイスと前提条件を確認する

この記事では、ユーザーインターフェイスにアクセスし、学習コース用の正しいフォルダープロファイルとベースパスを設定する方法について説明します。

## インターフェイスへのアクセスとナビゲーション

ユーザーインターフェイスにアクセスするには、次の手順を実行します。

1. AEM インスタンスにログインします。
2. AEM ナビゲーションページで、**ガイド**&#x200B;を選択します。
3. 現在、**Experience Manager Guidesのホームページ**&#x200B;にアクセスしています。 ナビゲーションスイッチャーを使用して、次のビューに切り替えます。

   - **ホーム**: Experience Manager Guidesへのログイン時に表示されるデフォルトページ。 フォルダーレベルのさまざまな設定を行うことができます。
   - **Editor**: Experience Manager Guidesでコースコンテンツを作成できる、使いやすいエディターです。
   - **マップコンソール**：コースの公開のすべての側面を処理するための専用のワークスペースを提供します。

   詳しくは、[Adobe Experience Manager Guides ホームページのエクスペリエンス &#x200B;](../user-guide/intro-home-page.md)を参照してください。

   ![](assets/aem-navigation-switcher.png){width="350"}

## 前提条件

ユーザーインターフェイスを使い始めるには、最初にExperience Manager Guides ホームページの&#x200B;**ユーザー環境設定**&#x200B;で正しい&#x200B;**フォルダープロファイル**&#x200B;と&#x200B;**ベースパス**&#x200B;を設定する必要があります。

![](assets/setup-folder-profile.png)

フォルダープロファイルは、オーサリングテンプレート、出力テンプレート、出力プリセット、その他のフォルダーレベルの設定を定義します。 Experience Manager Guidesでは、複数のフォルダープロファイルをサポートしています。管理者は、組織内の様々な部門や製品の設定を分離できます。 誤ったフォルダープロファイルを使用すると、テンプレートが欠落したり、機能が制限されたりする可能性があります。 使用するフォルダープロファイルが不明な場合は、続行する前に管理者にお問い合わせください。

ユーザー環境設定ページは、次の2つのタブで構成されています。

- **一般**: フォルダープロファイル、ベースパス、ルートマップなどを選択できます。
- **アピアランス**: アプリケーションのテーマと学習コンテンツのソースビューを選択するオプションを提供します。

詳しくは、Experience Manager Guidesの[&#x200B; ユーザー設定](../user-guide/intro-home-page.md#user-preferences)を参照してください。
