# クイズプロパティ

クイズプロパティは、次のオプションを使用して設定できます。

- クイズタイトルまたはクイズイントロにカーソルを置き、右側のパネルから&#x200B;**コンテンツのプロパティ** パネルを開きます。 クイズのプロパティはここから編集できます。

  ![](assets/quiz-properties.png){width="650"}

- クイズのオーサリング時に、クイズのタイトルと概要が空白のままになっていないことを確認します。公開されたクイズの最初のスライドに表示されます。 誤って削除した場合は、**クイズのタイトルを追加**&#x200B;および&#x200B;**クイズのイントロを追加** オプションを選択して復元できます。
- クイズの採点を有効にするには、**採点クイズ**&#x200B;切り替えをオンにします。 有効にすると、パーセント基準または絶対基準のグレーディング方法を選択し、それに応じて合格条件を定義できます。

  ![](assets/quiz-grading.png){width="350"}

- また、以下に示すように、回答提出オプション、フィードバックとレビューオプション、クイズのタイミング、レイアウト設定などの追加設定を行うこともできます。

  ![](assets/additional-quiz-properties.png){width="350" height="200"}

- クイズプロパティの「**コンテンツの再訪を許可する**」オプションを使用して、進行中のクイズ中にコースコンテンツを再訪できます。オープンブックテストと同様です。 ただし、このオプションが有効になっている場合、送信オプションを設定することはできず、クイズタイミング機能は無効になっています。
  ![](assets/quiz-allow-content-revist.png){width="350" height="200"}
