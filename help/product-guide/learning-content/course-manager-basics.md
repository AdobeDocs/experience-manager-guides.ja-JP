# コースマネージャーの基本を理解する

**コースマネージャー**&#x200B;は、コースを作成および管理するための中心的なワークスペースです。 新しいコースを作成すると、コースの構築を開始できるコースマネージャーパネルが自動的に開きます。

コースマネージャーを使用する場合は、次の点を考慮してください。

- コースが読み取り専用モードで開き、バージョン 1.0が自動的に割り当てられ、コースの初期バージョンが示されます。
- コースを編集するには、**オプション** メニューからロックを取得する必要があります。 コースがロックされると、コースに存在するトピックの追加や既存のトピックの編集を開始できます。

  ![](assets/course-manager-lock-course.png)
- パネルの&#x200B;**マップコンソール** アイコンを使用すると、マップコンソールに移動し、管理者が設定した出力プリセットが表示されます。 **オプション** メニューから&#x200B;**マップコンソール**&#x200B;にアクセスすることもできます。
