# クイズの概要

クイズは、コース作成者が学習者のエンゲージメントに対する評価を作成し、理解を追跡するのに役立ちます。 このセクションでは、コースにクイズを追加することから始まり、スコアリングルールやフィードバックなどを管理するために質問タイプを設定することから始まる、完全なクイズのワークフローについて説明します。
