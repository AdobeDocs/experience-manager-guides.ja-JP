# 製品トレーニングと学習コンテンツの概要

製品トレーニングと学習コンテンツのサポートにより、企業環境内でインタラクティブなe ラーニングコンテンツを簡単に作成および管理できます。 テンプレートを使用してコースを作成し、インタラクティブな要素（アコーディオン、カルーセル、マルチメディアなど）を追加し、異なる質問タイプを使用してクイズを追加したり、質問バンクを通じてコースを追加したり、サポートされている出力形式でコースを公開したりできます。

![](assets/learning-and-training-content-components-new.png)

## 主要な機能の一覧

主な機能は次のとおりです。

- 学習コンテンツの一元管理
- テンプレート駆動型オーサリング
- コンテンツの再利用
- クイズの作成と管理
- 業界をリードする翻訳管理
- web ベースのレビューワークフロー
- すぐに使えるSCORMおよびPDF出力フォーマットを使用したマルチチャネルパブリッシング
