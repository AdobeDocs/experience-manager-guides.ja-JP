# SCORM 出力プリセットの構成

プリセットを作成したら、SCORM プリセット設定を指定します。 プリセット設定オプションは、「一般」、「コンテンツ」および「公開」タブに整理されています。

- **一般：** サポートされているバージョン、出力パス、ZIP ファイル名、出力テンプレート、学習者のエクスペリエンスに関連するその他のオプションなど、基本的な出力設定を指定するために使用します。

  ![](assets/scorm-general-tab-v3.png){width="650" align="left"}

  **学習者のエクスペリエンス**

   - **学習者は、コンテンツを順番に進める必要があります**：学習者がクイズを一定の順序で進めることができるようになり、質問をスキップして前に進んだり、質問にジャンプしたりできなくなります。
   - **学習者は、すべての質問を先に進めるように試す必要があります**：学習者がクイズを送信する前にすべての質問を試み、不完全な送信を防ぐ必要があります。
   - **試行ごとの質問順序をランダム化**：試行ごとに異なる順序でクイズ質問を表示し、予測可能性を減らすのに役立ちます。
   - **各試行の回答の選択肢をランダム化**：すべての試行で各質問の回答オプションをシャッフルし、位置に基づいて推測される可能性を減らします。
   - **クイズレポートで質問 ID を使用**：クイズレポートに一意の質問 ID が含まれるので、結果の追跡、分析および特定の質問へのマッピングが容易になります。
   - **生成後ワークフロー**：このオプションを選択すると、すべての設定済みワークフローが含まれる新しい「生成後ワークフロー」ドロップダウンリストが表示されます。

- **コンテンツ：** 使用可能な条件付きフィルタリング（DITAVAL または一部の条件プリセットを使用）と変数セットの指定に使用します。

  ![](assets/scorm-content-tab.png){width="650" align="left"}

- **公開：** この設定は、出力を SCORM Cloud に公開して直接アクセスする場合にのみ使用します。

  ![](assets/scorm-publish-tab.png){width="650" align="left"}

すべての変更を設定したら、SCORM プリセット・ページのツールバーの右隅にある **保存** を使用して、SCORM プリセットの変更を保存します。
