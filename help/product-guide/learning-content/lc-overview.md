# 学習コンテンツの概要

Experience Manager Guidesの製品トレーニングおよび学習機能でサポートされているラーニングコンテンツまたは **トピック** を使用すると、e ラーニングコース内にテンプレート駆動型コンテンツを作成できます。 これにより、テキスト、ウィジェット、画像、マルチメディアなど様々な要素を組み込んで、学習体験を強化できます。
