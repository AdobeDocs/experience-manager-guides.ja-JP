# 学習コンテンツの概要

Experience Manager Guidesの製品トレーニングおよび学習機能でサポートされている学習コンテンツまたは&#x200B;**トピック**&#x200B;を使用すると、e ラーニングコース内でテンプレート駆動型コンテンツを作成できます。 テキスト、ウィジェット、画像、マルチメディアなどのさまざまな要素を組み込んで、学習体験を向上させることができます。
