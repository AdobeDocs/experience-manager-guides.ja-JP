# 検索パネル

エディターの検索パネルでは、検索語句や適用されたフィルターに基づいてコンテンツの編集時に表示されるファイルのサブセットにすばやくアクセスできるので、生産性が向上します。 トピックまたはマップにドラッグ&amp;ドロップするだけで、検索された 1 つまたは複数のファイルを簡単に開いたり、既存のファイル内で使用したりできます。 エディターの下部に **検索パネル** があります。

検索パネルには、次の場所からアクセスできます。

- **エディターインターフェイス**:**エクスプローラーパネル** から **検索アイコン** を選択するか、**コンテンツ編集領域** の左下隅にある **検索アイコン** を使用します。 詳しくは、[ エクスプローラーパネルからの検索 ](#search-from-the-explorer-panel) を参照してください。

  ![](images/search-icon-editor.png){align="left"}

- **ホームページ**：ホームページでリポジトリーインターフェイスから移動する際に **検索パネルに表示** オプションを使用します。 詳細表示については、[ リポジトリから検索 ](#search-from-repository-interface-on-the-home-page) してください。

  ![](images/search-panel-repo.png){align="left"}

## 主なメリット

- すべての検索結果を一元的に表示して、参照しやすくする。
- ドラッグ&amp;ドロップ機能により、現在のトピックまたはマップに直接参照を挿入できます。
- エディターを離れることなく検索を変更または絞り込むための柔軟なオプション。

## エクスプローラーパネルからの検索

エディターインターフェイスで作業する場合、ファイルのセットをフィルタリングして、必要な関連ファイルのサブセットを表示できます。 エクスプローラーからファイルを検索するには、次の手順を実行します。

1. **エクスプローラーパネル** 右上隅にある **検索** アイコンまたは **コンテンツ編集領域** の左下にある **検索** アイコンを選択します。 これにより、「**検索リポジトリ**」ダイアログが開き、ホームページのリポジトリインターフェイスと同じ検索およびフィルタリング機能が提供されます。

   >[!NOTE]
   >
   >現在のセッションで既に検索結果が存在する場合、エクスプローラーまたはコンテンツ編集領域の左下にある **検索アイコン** を選択すると、パネルが開き、その前の結果が表示されます。 検索を更新または絞り込むには、「**検索を修正**」を選択します。

   ![](images/search-repo-dialog.png){align="left"}

2. 検索を実行し、必要に応じてフィルターを適用します。 検索およびフィルターオプションの詳細な手順については、[ 検索とフィルターエクスペリエンス ](./home-page-repository-view.md#search-and-filter-experience) を参照してください。

3. 検索が完了したら、「**検索パネルに表示**」を選択します。 最近の検索が、エディターの下部にある検索パネルに表示されます。

   ![](images/show-search-explorer.png){align="left"}

4. 検索結果を更新するには、検索パネルで「**検索を変更**」オプションを選択し、新しい結果を得るために条件を更新します。

   ![](images/modify-search.png){align="left"}

検索結果が検索パネルに表示されたら、検索結果を操作できます。操作するには、1 つまたは複数のファイルをパネルから直接開いて編集するか、選択したファイルを既存のトピックまたはマップにドラッグ&amp;ドロップして参照を追加します。

![](images/edit-search-panel.png){align="left"}

## ホームページのリポジトリインターフェイスからの検索

ホームページで検索を実行してリポジトリインターフェイスにフィルターを適用する際に、「**検索パネルに表示**」を選択すると、エディターインターフェイスにリダイレクトされる。 すべての検索結果が、エディターインターフェイスの下部にある検索パネルにミラーリングされます。

検索パネルから、現在のトピックにファイルを **ドラッグ&amp;ドロップ** して、参照をシームレスに添付したり、複数のファイルを同時に編集したりできます。 さらに、「検索」パネルにある「**検索を修正**」オプションを使用して、検索結果を絞り込むことができます。

![](images/search-panel-from-home.png){align="left"}
