# エディターでのキーボードショートカット {#id2056B0A0WHT}

エディターには、キーボードショートカットを使用して実行できる多くの操作があります。 次の表に、これらのキーボードショートカットの一覧を示します。

>[!NOTE]
>
> キーボードショートカットの文字キーでは、大文字と小文字は区別されません。

## WindowsおよびmacOSのキーボードショートカット

| Web エディターでの操作 | Windows | macOS |
|-----------------------|-----------------|-----------------|
| 選択したテキストに太字の書式を適用する | ***Ctrl***+***B*** | ***コマンド***+***B*** |
| 選択したテキストに斜体の書式設定を適用 | ***Ctrl***+***I*** | ***コマンド***+***I*** |
| 選択したテキストに下線書式を適用する | ***Ctrl***+***U*** | ***Command***+***U*** |
| 左側のパネルにフォーカス | **Ctrl***+**Shift**+***1*** | ***Command***+**Shift**+***1*** |
| 右側のパネルにフォーカス | **Ctrl***+**Shift**+***2*** | ***Command***+**Shift**+***2*** |
| 現在のエディターに集中 | **Ctrl***+**Shift**+***0*** | ***Command***+**Shift**+***0*** |
| 作成者ビューまたはSource ビューでドキュメントを保存 | ***Ctrl***+**Shift**+***S*** | ***Command***+**Shift**+***S*** |
| 開いているすべてのドキュメントを作成者またはSource ビューで保存 | ***Ctrl***+***S*** | ***Command***+***S*** |
| 挿入エレメントを表示ダイアログ | ***Alt***+***1*** | ***オプション***+***入力*** |
| 選択したエレメントの前にエレメントを挿入するには、エレメントの挿入ダイアログを表示します | ***Alt***+***Shift***+***Enter*** | ***オプション***+***Shift***+***Enter*** |
| カーソルを1単語ずつ左に移動します。 | ***Ctrl***+***左向き矢印*** | ***オプション***+***左向き矢印*** |
| カーソルを1単語ずつ右に移動します。 | ***Ctrl***+***右向き矢印*** | ***オプション***+***右向き矢印*** |
| カーソルを現在の行の先頭に移動します。 | ***ホーム*** | ***Command***+***左向き矢印*** |
| カーソルを現在の行の最後に移動します。 | **終了** | ***Command***+***右向き矢印*** |
| カーソルを次の要素の先頭に移動します。 | ***タブ*** | ***タブ*** |
| リスト項目を次のレベルにインデントします。 | ***タブ*** | ***タブ*** |
| カーソルを前のエレメントの最後に移動します。 | ***Shift***+***Tab*** | ***Shift***+***Tab*** |
| リスト項目を前のレベルにインデントします。 | ***Shift***+***Tab*** | ***Shift***+***Tab*** |
| Source ビューでのXMLの自動インデント | ***Ctrl***+***Alt***+***L*** | ***Command***+***Option***+***L*** |
| 現在の要素を分割 | ***入力*** | ***Return*** |
| 選択したテキストをコピー | ***Ctrl***+**C** | ***コマンド***+**C** |
| 貼り付け | ***Ctrl***+**V** | ***コマンド***+**V** |
| 書式をペーストして一致 | ***Ctrl***+***Shift***+**V** | ***Command***+***Shift***+**V** |
| 印刷ダイアログを開く | ***Ctrl***+**P** | ***コマンド***+**P** |
| 最後の操作を取り消す | ***Ctrl***+**Z** | ***Command***+**Z** |
| 最後のアクションをやり直す | ***Ctrl***+**Y** | ***Command***+**Y** |
| 非改行スペース `(&nbsp;)`を挿入 | ***Alt***+**スペース** | ***オプション***+**スペース** |
