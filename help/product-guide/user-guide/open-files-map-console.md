# マップコンソールでファイルを開く

次の手順を実行して、マップコンソールでDITA マップファイルを開きます。

1. ホーム ページから&#x200B;**マップ コンソール**&#x200B;を開きます。

   ![新規](images/map-console-home-page.png){width="800"}

2. マップファイルが選択されていないので、マップ管理機能と公開機能を使用するマップファイルを選択するように求められます。

   ![新規](images/empty-screen-map-console.png)

3. **マップを選択**&#x200B;し、DITA マップファイルが配置されているパスを選択します。

   マップファイルがマップコンソールで開きます。 デフォルトでは、**出力プリセット** タブが選択されています。

   ![新規](images/map-console-screen.png){width="800"}

   >[!NOTE]
   >
   >  マップコンソールで開いたマップは、エディターで使用可能なマップビューと同期されます。

## エディターからマップファイルを開く

既存のマップファイルをマップコンソールでエディターから開くこともできます。

1. リポジトリビューで、DITA マップファイルに移動して選択します。

   マップファイルがマップビューで開きます。

2. 「**マップコンソールで開く**」アイコンを選択します。

   マップファイルがマップコンソールで開きます。

   ![新規](images/map-console.png)
