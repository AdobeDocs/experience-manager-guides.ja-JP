# Adobe Experience Managerでのメタデータの操作 {#id21BJ00QD0XA}

Adobe Experience Manager Assetsには、アセットを作成してメタデータを追加する機能が用意されています。 あらゆるアセットにメタデータを保持できます。 アセットのメタデータを管理したり、メタデータにもとづいてアセットを自動的に整理および処理したりできます。

Adobe Experience Managerでのメタデータ管理について詳しくは、[ デジタルアセットのメタデータの管理](https://experienceleague.adobe.com/docs/experience-manager-65/assets/using/metadata.html?lang=ja)を参照してください。
