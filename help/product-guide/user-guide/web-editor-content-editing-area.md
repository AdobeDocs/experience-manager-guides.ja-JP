# エディターのコンテンツ編集領域

>[!INFO]
>
> このトピックは、新しいエディターと古いエディターの両方に適用されます。 コア機能は一貫していますが、ユーザーインターフェイス、用語、インタラクションの違いは、該当する場合はタブとコールアウトを使用してコンテンツ内に示されます。

コンテンツ編集エリアは、トピックまたはマップのコンテンツが表示される場所です。 この領域ですべてのコンテンツを編集します。 WYSIWYGで編集中のコンテンツが表示されます。

コンテンツ編集領域の左下に、現在のカーソル位置にエレメントのパンくずリストがあります。 右下隅に、使用可能なエディタービューが表示されます。

>[!NOTE]
>
> マップファイルで作業している場合は、コンテンツ編集領域にさまざまなオプションまたはビューが表示されます。 詳しくは、[&#x200B; マップエディター機能](./map-editor-advanced-map-editor.md)を参照してください。
