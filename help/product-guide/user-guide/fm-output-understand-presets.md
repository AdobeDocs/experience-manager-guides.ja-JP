# 出力プリセットについて {#understanding_output_settings}

Adobe Experience Manager Guidesでは、PDF、HTML5、およびEPUB形式でのFrameMaker ドキュメントの出力作成をサポートしています。 これらの出力形式を使用して、様々な出力プリセットを設定できます。 出力プリセットは、コンテンツを公開するカスタマイズされた出力形式を表します。
