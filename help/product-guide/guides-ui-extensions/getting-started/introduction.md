# AEM Guides拡張機能パッケージ

AEM Guides拡張機能パッケージを使用すると、必要に応じてアプリをカスタマイズできます。 ツールバーに別のボタンを追加する コメント ボックスに新しい入力フィールドを追加しますか？ コンテキストメニューに別のオプションがありますか？ 今ではすべてを実行できます。

このパッケージの使用方法については、[ インストールガイド ](./integrating-customisations.md)を参照してください

> **_NOTE:_**&#x200B;このAEM Guides拡張機能パッケージは、AEM Guides バージョン >= 4.3 （オンプレミス）および>= 2310 （クラウド）でサポートされています
