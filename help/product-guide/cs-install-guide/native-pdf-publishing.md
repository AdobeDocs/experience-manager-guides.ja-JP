# PDFとのネイティブな連携

AEM Guidesでは、PDF出力のネイティブ生成機能を使用して、個々のトピックのPDFやマップファイルを生成できます。 PDF出力の生成時に、PDF テンプレートをカスタマイズして使用できます。 ページレイアウトを作成およびデザインして、必要に応じてPDF出力の様々なセクションを表示できます。

さらに、AEM Guidesでは、ローカライズされた文字列を定義するのに役立つ言語変数もサポートしています。 また、PDF出力に含まれる様々な要素のスタイルを定義することもできます。 上級ユーザーの場合は、CSSを直接操作して高度なページレイアウトを作成することもできます。
