# 既存コンテンツの移行 {#id181AGB000Y4}

>[!TIP]
>
> 既存のコンテンツの移行とアップロードの使用に関するベストプラクティスについては、ベストプラクティスガイドの&#x200B;*コンテンツの移行とアップロード*&#x200B;の節を参照してください。
