# AEM Guidesを使用したコンテンツの作成 {#id221EB0P0E20}

AEM Guidesには、3つの標準エディターが搭載されています。 ドキュメントの作成、編集、公開に使用できます。

1. **Web エディター**：構造化ドキュメントの作成と編集に使用します。
1. **マップエディター**: DITA マップおよびブックマップファイルの作成と編集に使用します。
1. **Ditaval Editor**：条件付き出力の生成に使用します。
