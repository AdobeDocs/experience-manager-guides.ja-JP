# Adobe Experience Manager Guidesについて {#id1645E0A50MY}

Adobe Experience Manager Guides \（このガイドでは&#x200B;*AEM Guides*&#x200B;と呼んでいます）は、強力なエンタープライズグレードのコンポーネントコンテンツ管理ソリューション \（CCMS\）です。 オンプレミスとクラウドサービスの2種類があります。 Adobe Experience Managerでネイティブ DITA サポートを有効にし、AEMがDITA ベースのコンテンツの制作と配信を処理できるようにします。 使いやすい組み込みweb エディターを使用して、コンテンツを制作できます。

AEM Guidesは、DITA コンテンツのオーサリング、コラボレーション、レビュー、翻訳、検索、レポート、メタデータ管理など、CCMSの中核機能を提供します。これにより、コンテンツの効率的な再利用と強力なワークフローにより、制作者は短時間でより多くの作業をおこなうことができます。 さらに、クラス最高のシングルクリック公開機能を活用して、最も一般的なフォーマット（Adobe Experience Manager Sites、PDF、HTML5、EPUB、DITA-OTを通じたカスタム出力）用のDITA ベースの出力を生成できます。

AEM Guidesなら、パーソナライズされたシームレスなエクスペリエンスをエンドユーザーに提供し、購入前と購入後のコンテンツの一貫性と統一性を確保できます。 コンテンツの整合性を容易に詳細に制御できます。 さらに、ローカライズにかかる時間とコストも大幅に削減されます。

## 概要と利点

- エンドユーザー向けに購入前と購入後のコンテンツ体験を統一

- Experience Manager Sites、PDF、HTML5、EPUB、DITA-OTを通じたカスタム出力へのオムニチャネルのパブリッシングエクスペリエンス

- 既存のAdobeツールやシステムに関する知識と、単一のパートナーとの統合の可能性

- マーケティングコンテンツとテクニカルコンテンツをエンドツーエンドで管理する、単一のCMS

- コンテンツの効率的な再利用による市場投入の迅速化

- 強力なレビュー、コラボレーション、翻訳ワークフロー

- ローカライゼーションに要する時間とコストを削減

- 保守に要する負担の軽減
