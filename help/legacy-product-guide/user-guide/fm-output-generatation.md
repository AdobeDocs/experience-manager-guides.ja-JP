# FrameMaker ドキュメントの出力の生成 {#id1645H8040HD}

AEM Guides 1.1以降では、AEM リポジトリで利用可能なFrameMaker ドキュメント \（`.book`および`.fm`\）を公開することもできます。 ブックファイルにDITA文書とFrameMaker文書の組み合わせが含まれている場合は、AEM Guidesを使用して、そのような文書も公開できます。 FrameMaker ドキュメントは、次の形式で公開できます。

- PDF

- HTML5

- EPUB


作成者は、いくつかのリンクをクリックするだけで、出力が生成されます。 ブックファイル全体の出力を生成することも、個々のFrameMaker ファイルを選択して公開することもできます。 出力を生成する前に、FrameMaker ファイルを変更することもできます。 AEM リポジトリで.fmまたは.book ファイルを選択すると、ツールバーの「**FrameMakerで開く**」ボタンが表示されます。 このボタンをクリックすると、選択したファイルがFrameMakerで開きます。

次の節は、出力生成ワークフローを使用してFrameMaker ドキュメントを公開するパブリッシャー向けです。

- [出力プリセットについて](fm-output-understand-presets.md#)

- [.book ファイルまたは.fm ファイルの出力を生成](fm-output-generate.md#)
