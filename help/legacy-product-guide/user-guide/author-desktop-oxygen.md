# 酸素の操作 {#id221BH0V0UE8}

Oxygen XML Author には、使いやすいオーサリング機能が付属しており、使いやすいインターフェイスを提供しています。

同じソースを使用して、PDF、ePub、HTMLおよびその他の多くの形式で出力を生成できます。

*環境内でこの機能を有効にするには、カスタマーサクセスチームにお問い合わせください。 これは、標準のサポートの一部として有効になっていません。*
