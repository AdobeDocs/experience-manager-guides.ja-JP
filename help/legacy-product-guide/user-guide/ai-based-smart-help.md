# AIを活用したスマートヘルプでコンテンツを検索



Experience Manager Guidesは、生成AI ベースのスマートヘルプを提供します。これは、[Adobe Experience Manager Guides ドキュメント &#x200B;](https://experienceleague.adobe.com/ja/docs/experience-manager-guides/using/overview)から関連コンテンツを見つけるのに役立つ会話型検索機能です。
質問に答えたり、有益な方法で回答を得たりすることができます。 製品ドキュメントのコンテンツに基づいて、クエリに対する回答を返します。 検索エンジンは完全に対話型です。 質問することができ、それに基づいて、さらに質問することもできます。 応答には、ソースドキュメントへのリンクも含まれています。詳細については、こちらを参照してください。

例えば、Experience Manager Guidesでドキュメントのトピックを作成することができます。 *トピックの作成方法を尋ねることができます。* 回答と関連記事へのリンクが表示されます。 次に、ドキュメントのPDF出力を生成する方法を学習する場合は、その方法について質問することができます。 例えば、*トピックをPDFに公開する方法を教えてください。* または&#x200B;*トピックのPDF出力を生成する方法を教えてください。*



Web エディターを開くと、**スマートヘルプ** パネルが右側に表示されます。



>[!NOTE]
>
> 管理者は&#x200B;**スマートヘルプ**&#x200B;機能を設定する必要があります。 詳しくは、「Cloud Servicesのインストールおよび設定ガイド」の「[AIを活用したスマートヘルプを設定してコンテンツを検索する](/help/product-guide/cs-install-guide/conf-smart-help.md)」セクションを参照してください。

![&#x200B; スマートヘルプパネル &#x200B;](images/smart-help-panel.png){width="300" align="left"}

***スマートヘルプ**&#x200B;パネルを表示します。*

会話型検索を使用して適切なコンテンツを検索し、クエリを解決するには、次の手順を実行します。

1. 「**スマートヘルプ** ![&#x200B; スマートヘルプアイコン &#x200B;](images/smart-help-icon.svg)」を選択して、パネルを開きます。



   >[!NOTE]
   >
   > [&#x200B; グローバルまたはフォルダーレベルのプロファイル &#x200B;](/help/product-guide/cs-install-guide/conf-folder-level.md#conf-ai-guides-assistant)で、管理者はパネルに表示されるデフォルトの質問を定義する必要があります。

1. 質問を入力して、関連するコンテンツをExperience Manager Guides ドキュメントで見つけます。 パネルでデフォルトの質問を選択するか、テキストボックスに質問を入力します。

1. **送信** ![送信アイコン &#x200B;](images/send-icon.svg)を選択するか、**Enter**&#x200B;を押して、質問に対する回答を表示します。

   質問に応じて、コンテンツ、該当する画像、記事へのリンクを表示できます。

   ![&#x200B; スマートヘルプパネルの応答](images/smart-help-panel-response.png){width="300" align="left"}


   *サンプルの質問を選択し、回答の内容と画像を表示します。*





1. 最後の記事へのリンクを選択し、質問に関する詳細情報を表示します。


1. **会話を消去** ![会話を消去](images/clear-conversation-icon.svg)を選択して、パネルから会話の履歴を削除します。 会話を始め、関連性の高いコンテンツを見つけることができます。

このスマート機能により、解決策をすばやく見つけ出し、ドキュメント作成に集中して、タスクを効率的に完了できます。
