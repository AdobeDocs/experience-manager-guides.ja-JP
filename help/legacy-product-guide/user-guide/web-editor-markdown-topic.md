# Web エディターからMarkdown ドキュメントを作成する {#id223MIE0B079}

Markdownは、プレーンテキストドキュメントに書式要素を追加するのに役立つ、軽量なマークアップ言語です。 AEM Guidesには、Web エディターからMarkdown \（.md\）トピックを作成、作成、プレビューする機能が用意されています。 既存のMarkdown ドキュメントをアップロードして、Web エディターで編集することもできます。

## Markdown トピックの作成

Web エディターからMarkdown トピックを作成するには、次の手順を実行します。

1. リポジトリビューでフォルダーを選択します。
1. 次のいずれかの方法で&#x200B;**Markdown トピック**&#x200B;を作成します。
   1. リポジトリパネルの右上隅にある「**DITA ファイルを作成**」アイコンをクリックし、ドロップダウンから「**Markdown Topic**」を選択します。

   ![](images/create-markdown-dita-topic.png){width="550"}

   1. 選択したフォルダーのオプションメニューから「**作成**」を選択し、ドロップダウンから「**マークダウントピック**」を選択します。

   ![](images/create-markdown-options-menu.png){width="550"}

1. 「**新規マークダウン トピックを作成**」ダイアログに、マークダウン トピックの名前とパスを入力します。

![](images/create-markdown-dialog.png){width="350"}

1. 「**作成**」をクリックします。

選択したパスにMarkdown トピックが作成され、編集用に開かれます。

## Markdown トピックの作成とプレビュー

Web エディターからMarkdown トピックを作成するには、次の手順を実行します。

1. リポジトリビューで既存のMarkdown トピック\（.md\）を選択します。
1. トピックをダブルクリックして、編集ウィンドウで開きます。

   Source モードでトピックが開きます。

1. Markdown トピックのコンテンツを更新します。
1. 右上の&#x200B;**プレビュー**&#x200B;をクリックして、トピックをプレビューします。

   >[!NOTE]
   >
   > プレビューモードは、ユーザーがブラウザーでトピックを表示したときに、トピックがどのように表示されるかを示します。

1. メインツールバーの&#x200B;**新しいバージョンとして保存**&#x200B;を使用して、トピックで行った変更を保存し、トピックの新しいバージョンを作成します。 メインツールバーの&#x200B;**すべて保存**&#x200B;を使用して、開いているすべてのトピックの変更を保存することもできます。

1. 右上の&#x200B;**閉じる**&#x200B;をクリックして、Markdown トピックを閉じます。

>[!NOTE]
>
> 変更を保存していない場合は、ポップアップが表示され、トピックを閉じる前に保存するように求められます。
